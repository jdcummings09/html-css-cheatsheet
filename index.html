HTML/CSS Cheatsheet

HTML Cheatsheet

Tag/Attribute Name Example Description
<header></header> <header> [child tags and content] </header> Semantic way to section off content at the top of the page. Can be used to hold navigation, titles, etc.
<main></main> <main> [child tags and content] </main> Semantic way to section off main content on the page. Can be used for articles, data, media, etc.
<footer></footer> <footer> [child tags and content] </footer> Semantic way to section off content at the bottom of the page. Can be used for contact info, about me links, etc.
<link> <link rel="stylesheet" href="style.css"> Used in the head section to link stylesheet to the html document.
<img> <img src="image.jpeg" alt="Description of image." width="150px" height="100px"> Used to add images and photos to html document.
<video> <video width="1920px" height="1080px" controls>
<source src="video.mp4" type="video/mp4">

</video>		 Used to add videos to html document.

CSS Cheatsheet

Tag/Attribute Name Example Description
color color: white; Will change text color to white for selected element(s).
background-color background-color: white; Will change the backgrount color to black for the selected element(s).
background-image background-image: url(./media/photos/pattern.jpeg); Will change the background image of the selected element(s) to the given image.
font-family font-family: Helvetica; Changes the font style/family of the selected element(s).
font-size font-size: 100px; Changes the font size of the seleced element(s).
font-weight font-weight: bold; Changes the font weight (thickness) of the selected element(s).
text-align text-align: center; Aligns the text of the selected element(s).
